# Daily Session OHLCV Comparison Report

FX/CFDのOHLCV時系列データから、同じ時間帯の値動きを日別比較してPDF化するツールです。

- `yfinance` から直接取得して実行可能（CSV不要）
- 必要ならCSV入力も利用可能
- 複数時間足を同時に比較し、ページ分割でPDF保存

## ファイル構成

- `main.py`: CLIエントリポイント
- `loader.py`: CSV読み込み・文字コード判定
- `resample.py`: 分足変換
- `plotter.py`: サブプロット描画（ローソク足、統計、出来高）
- `pdf_report.py`: PDFページング出力
- `requirements.txt`: 依存

## 想定CSV形式

CSVを使う場合は、少なくとも以下列が必要です。

- `datetime`（または `time`, `timestamp`, `date`, `約定日時`, `日時`）
- `open`
- `high`
- `low`
- `close`
- `volume` は任意

文字コードは UTF-8 / CP932 / Shift_JIS などを自動判定します。

## 使い方

### 1) yfinanceのみで実行（CSV不要）

```bash
python main.py \
  --ticker JPY=X \
  --days 20 \
  --session-start 09:00 \
  --session-end 10:30 \
  --timeframes 1 5 15 \
  --timezone Asia/Tokyo \
  --rows 2 \
  --cols 2 \
  --show-volume \
  --output out/session_report.pdf
```

### 2) CSVで実行

```bash
python main.py \
  --input data/ohlcv.csv \
  --symbol USDJPY \
  --start-date 2026-02-01 \
  --end-date 2026-03-01 \
  --session-start 09:00 \
  --session-end 10:30 \
  --timeframes 1 5 15 \
  --timezone Asia/Tokyo \
  --output out/session_report.pdf
```

### 3) 日付を明示指定

```bash
python main.py \
  --ticker EURUSD=X \
  --dates 2026-03-01 2026-03-05 2026-03-06 \
  --session-start 09:00 \
  --session-end 10:30 \
  --timeframes 1 5 15
```

## 主な引数

- `--input`: 入力CSV（省略可）
- `--ticker`: yfinanceティッカー（`--input` 省略時に利用）
- `--symbol`: 表示用シンボル名
- `--start-date`, `--end-date`: 対象日レンジ
- `--days`: 直近N営業日
- `--dates`: 日付を明示指定（空白区切り）
- `--session-start`, `--session-end`: 比較時間帯
- `--timeframes`: 比較時間足（例: `1 5 15`）
- `--timezone`: タイムゾーン（例: `Asia/Tokyo`）
- `--output`: PDF出力先
- `--rows`, `--cols`: 1ページ内サブプロット行列
- `--show-volume`: 出来高表示
- `--style`: matplotlibスタイル
- `--orientation`: `portrait` or `landscape`
- `--fixed-ymin`, `--fixed-ymax`: 固定価格スケール

## 注意点

- `yfinance` の1分足は取得可能期間に制限があります。長期間は5分足へフォールバックします。
- 指定時間帯にデータがない日は「データなし」と表示して処理継続します。
- 夜間セッション（例: `22:00-01:00`）にも対応しています。
